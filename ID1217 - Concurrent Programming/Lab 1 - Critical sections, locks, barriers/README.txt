README PalindromicWordFinder

Compile:
javac PalindromicWordFinder.java

Run:
Java PalindromicWordFinder [number of worker threads]
Where the only parameter is the number of worker threads. If none supplied, the program will use 8 as default.

Needed files:
"words.txt" (the list of words to check) needs to be in the same directory as the program. 
Make sure you have permissions to read from such file.

Output:
The program will make output to standard out and also write the list of palindromes to a file named "palindromes.txt".
Make sure that you have permissions to create and modify such file.
